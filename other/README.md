### scaffolding

create base of project in production server, somethings like this

```
project
│
├── app
│   ├── backend
│   └── client
├── etc
│   ├── cheatsheet
│   │   ├── elasticsearch
│   │   ├── init
│   │   ├── nginx
│   │   └── python
│   └── nginx
│       └── sites-available
└── var
    └── log
        └── marketing
```

args:

- `project`: project name
